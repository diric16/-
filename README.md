### 核心思路
1. **读取数据**  
   将时间序列数据分为三个时间段：  
   - 7-8 点  
   - 8-9 点  
   - 9-10 点  

2. **读取地理栅格数据**  
   将地理栅格数据作为背景地图，转换为 `base64` 编码，便于嵌入到 HTML 中。  

3. **使用 Plotly 创建交互式地图**  
   添加时间段切换按钮，用于切换不同时间段的数据。  

4. **支持交互功能**  
   - 支持缩放  
   - 支持平移  
   - 支持悬停查看详情  

5. **可视化设计**  
   - 不同时间段的数据点使用不同颜色标记。  
   - 点击某个时间段按钮，显示该时间段的对应数据点。  

6. **输出结果**  
   生成一个 HTML 文件，用于展示交互式可视化结果。
